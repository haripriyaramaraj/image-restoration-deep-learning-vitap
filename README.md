# Image Restoration using Deep Learning  
**Vellore Institute of Technology (VIT) | Mar 2023 – May 2023 | Individual Research Project**

## Project Overview
Developed a CNN-based image restoration model to remove real-world noise from images using the SIDD (Smartphone Image Denoising Dataset) benchmark. The model significantly improves image quality for downstream computer-vision and data-analysis tasks.

## Key Achievements
- Achieved **40% improvement** in image quality metrics (PSNR) over baseline
- Built end-to-end pipeline for data preprocessing, model training, and evaluation
- Demonstrated strong skills in Deep Learning, image data handling, and performance optimization

## Technologies Used
- **Python**, TensorFlow, Keras, OpenCV
- NumPy, Matplotlib, Pandas (for data analysis & visualization)
- Google Colab / Local GPU environment

## Methodology
1. **Data Preparation**: Loaded and pre-processed SIDD dataset (noisy-clean image pairs)
2. **Model Architecture**: Custom CNN with residual blocks for noise removal
3. **Training**: Adam optimizer, MSE + SSIM loss, 50 epochs with early stopping
4. **Evaluation**: PSNR, SSIM, and visual comparison

## Results
| Metric          | Baseline | Proposed Model | Improvement |
|-----------------|----------|----------------|-------------|
| PSNR (dB)       | 28.4     | 39.7           | **+40%**    |
| SSIM            | 0.82     | 0.94           | +14.6%      |
